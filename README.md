# Expense Tracker

A simple Python-based expense tracker that allows users to add expenses
and view their spending records stored in a JSON file.

## Features
- Add new expenses
- View all expenses
- Store data in JSON format
- Simple and easy to use

## Technologies Used
- Python 3.12
- JSON
- Git & GitHub
